OpenGL Kinect - Bowls (Auditory/Visual Cueing Game)

A bowls (Lawn Bowls) game in 3D using the Microsoft Kinect 1 for input. It is rendered using a closed source OpenGL wrapper. It was made by a team of 4, 2 focusing on the design and paperwork and 2 (me and another) working on the Game Code. My teammate Jamie is the player in the demonstration video.

The assignment required research into topics such as:

  • Kinect Input
  • 3D Graphics

Star Fire: Allegro 4 Game

In this assignment I needed to make a game of my choosing (with a loose set of requirements) in the Allegro 4 graphics library. It also had to demonstrate object orientated techniques such as Polymorphism, Inheritance, Composition and Encapsulation. I decided to make an arcade space shooter, as I wanted to mainly focus on the code rather than graphics and level design.

Skills acquired:

  • Object Orientated Programming
  • Basic UML
  • Working with a Graphics API
